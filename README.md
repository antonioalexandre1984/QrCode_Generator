## ⚙️ installation
```
# Open a terminal and copy this repository with the command
$ git clone (https://github.com/antonioalexandre1984/buzzvel.git)
```

```
# Access the application folder
$ cd buzzvel

# install the dependencies
$ yarn

# Start the application
$ yarn dev

```
